# СкладПро — FAQ и регламент по интеграции с 1С

## 1. Общие вопросы о продукте

**Что такое СкладПро?**
Облачный сервис складского учёта для небольших производств и оптовых компаний. Учёт остатков, партий, сроков годности, автоматическое планирование закупок и производства.

**Кому подходит?**
Мебельным, строительным и продуктовым производствам и оптовикам со штатом от 3 до 150 сотрудников и объёмом от 50 SKU.

**Есть ли мобильное приложение?**
Да, приложение для сборщиков и кладовщиков (iOS/Android) — сканирование штрихкодов, приёмка, отгрузка.

**Можно ли работать без интернета?**
Мобильное приложение кэширует последние данные и синхронизируется при появлении сети. Веб-версия требует подключения.

## 2. Интеграция с 1С

**С какими версиями 1С работает интеграция?**
- 1С:Управление торговлей 11
- 1С:Бухгалтерия 3.0
- 1С:ERP (частичная поддержка, уточняется индивидуально)

**Как происходит синхронизация?**
Через модуль-коннектор (устанавливается на сторону 1С) + REST API СкладПро. Синхронизация остатков, номенклатуры и документов — каждые 15 минут, либо по кнопке "Синхронизировать сейчас".

**Регламент подключения (шаги для оператора/клиента):**
1. Клиент присылает версию 1С и релиз конфигурации.
2. Мы высылаем файл коннектора (.epf) и инструкцию по установке.
3. Клиент устанавливает коннектор в 1С (обычно занимает 15–20 минут, делает штатный 1С-специалист клиента или наш инженер удалённо за отдельную плату).
4. Настраивается сопоставление справочников (номенклатура, склады, контрагенты).
5. Тестовая синхронизация на 5-10 позициях.
6. Полная синхронизация, включение автоматического режима.

**Средний срок настройки:** 1–3 рабочих дня после получения доступа к 1С.

**Типовая проблема: "Остатки не сходятся с 1С"**
Топ-3 причины (в порядке частоты):
1. Синхронизация не запускалась после последних изменений в 1С — решение: нажать "Синхронизировать сейчас" вручную.
2. Товар заведён в 1С и в СкладПро под разными артикулами — решение: проверить сопоставление номенклатуры в разделе "Настройки → Сопоставление".
3. Расхождение из-за ручных корректировок в 1С, сделанных в обход коннектора — решение: включить логирование и прислать нам период расхождения, разберём вручную.

**Что делать, если самостоятельно не разобрались?**
Прислать: (1) скриншот остатков в 1С, (2) скриншот остатков в СкладПро по той же позиции, (3) дату последней успешной синхронизации. Передать в тех.поддержку с пометкой "1С — расхождение остатков".

## 3. Учёт партий и сроков годности

**Как включить учёт по партиям?**
Настройки → Склад → "Партионный учёт" → включить. После включения все новые приёмки будут требовать указания партии и срока годности.

**Можно ли получить отчёт по товарам с истекающим сроком годности?**
Да, отчёт "Контроль сроков годности" в разделе Отчёты. Настраивается порог предупреждения (например, за 14 дней до истечения).

**Списание просроченного товара — как оформить?**
Документ "Списание" с указанием причины "Истёк срок годности". Автоматически формирует проводку по себестоимости списанной партии.

## 4. Демо-доступ

**Условия предоставления демо:**
- Бесплатный доступ на 14 дней, полный функционал, без ограничений по числу пользователей.
- Демо-база предзаполнена тестовыми данными (можно попросить чистую базу для загрузки своих данных).
- Продление демо возможно один раз на 7 дней по запросу.
- Для оформления демо нужны: название компании, сфера деятельности, примерный объём SKU, контактное лицо.
